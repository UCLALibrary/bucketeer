CSV Upload

The CSV upload form can be used to upload CSV files for processing by the Bucketeer application. Looking for a file to upload? UCLA's CSV files can be found in the Eureka GitHub repository.

Process previous failures only

Instructions

To start a batch job, upload a CSV file (like one of those found in the Eureka GitHub repository). If this is the first time that the file has been processed, do not select the Process previous failures only option -- this is only for files that have been previously processed (and that have had failed images).

To submit the form, you will also need to input your Slack handle so that Bucketeer can notify you when your batch job is done. To be notified of this, you need to join the #bucketeer-jobs Slack channel. The processed CSV file will also be uploaded to you there.
