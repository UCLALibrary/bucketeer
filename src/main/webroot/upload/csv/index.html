CSV Upload

The CSV upload form can be used to upload CSV files for processing by the Bucketeer application.

Process previous failures only
Remember to join the #bucketeer-jobs Slack channel to receive notification of the batch job's completion.

Instructions

To start a batch job, upload a CSV file (like one of those found in the Eureka GitHub repository). If this is the first time that the file has been processed, do not select the Process previous failures only option -- this is only for files that contain items that failed to be processed in a first run.

There are only two required fields in the uploaded CSV file: 'Item ARK' and 'File Name'. The Bucketeer ingest process will add two additional fields to the output CSV file: 'Bucketeer State' and 'IIIF Access URL'. Do not remove them. They will be needed if the CSV file is later re-ingested into Bucketeer (for instance, to re-process items that failed on the first run).

You will need to provide your Slack handle so that Bucketeer can notify you when your batch job is done. You must also have joined the #bucketeer-jobs Slack channel to see that notification. The processed CSV file will be uploaded to you there. You do not need to include the '@' symbol before your Slack handle.
